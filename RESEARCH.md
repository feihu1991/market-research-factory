# 市场调研 & 产品工厂

## 工作流程

```
1. 调研阶段
   ↓
2. 分析痛点
   ↓
3. 选择机会
   ↓
4. 构建MVP
   ↓
5. 输出产品
```

## 1. 调研 Prompt

```
请使用 Last 30 Days 技能研究 [主题]。

在过去30天里，人们在Reddit和X上对[主题]有什么抱怨？

整理成：
- Top 痛点 (按频率排序)
- 具体抱怨和功能请求
- 现有方案的不足
- 新产品机会
```

## 2. 痛点列表 (Pain Points)

| # | 痛点 | 频率 | 机会 |
|---|------|------|------|
| 1 | Setup困难 | 高 | 配置向导工具 |
| 2 | 发现技能难 | 中 | 技能搜索 |
| 3 | 成本问题 | 中 | 便宜模型替代 |

## 3. 产品创意

| 痛点 | MVP方案 |
|------|---------|
| Setup困难 | 向导式配置工具 |
| 发现技能难 | 技能推荐引擎 |
| 成本问题 | 模型对比工具 |

## 4. 产品模板

### 产品1: OpenClaw Setup Wizard
- 功能: 一步步引导配置OpenClaw
- 技术: 单页HTML
- 目标: 降低入门门槛

### 产品2: Skill Discovery
- 功能: 按需求搜索合适技能
- 技术: 关键词匹配 + AI推荐

### 产品3: Model Cost Calculator
- 功能: 计算不同模型使用成本
- 技术: 简单的计算器
